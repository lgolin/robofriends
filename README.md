# :robot: RoboFriends

A cool modern search card app.

## :heavy_check_mark: Prerequisites

NPM installed

## :checkered_flag: Getting Started

On terminal:

1. Run `git clone https://github.com/lgolin/robofriends.git`
2. Run `cd robofriends`
3. Run `npm install`
4. Run `npm start`

- HTML
- CSS

- React.js
