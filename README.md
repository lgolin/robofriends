# RoboFriends

A cool modern search card app.

## Prerequisites

NPM installed

## Getting Started

On terminal:

1. Run `git clone https://github.com/lgolin/robofriends.git`
2. Run `cd robofriends`
3. Run `npm install`
4. Run `npm start`

## Built With

- HTML
- CSS
- Javascript
- React.js
